# Repository for "Elevated microbial gene flow in the inflammatory human gut"

Welcome to the official code repository for the manuscript titled **"Elevated microbial gene flow in the inflammatory human gut."** This repository contains the code and resources required to reproduce the main analyses, figures, and supplementary results presented in the paper.

## Repository Structure

This repository is organized into several folders corresponding to different parts of the analysis and manuscript results:

- **Data-preprocessing/**  
  Scripts and utilities for preprocessing raw sequencing data, metadata, and intermediate files used in the study.

- **Figure-01/**  
  Code and resources required to reproduce Figure 1 from the manuscript.

- **Figure-02/**  
  Code and resources required to reproduce Figure 2 from the manuscript.

- **Figure-03/**  
  Code and resources required to reproduce Figure 3 from the manuscript.

- **Figure-04/**  
  Code and resources required to reproduce Figure 4 from the manuscript.

- **Figure-05/**  
  Code and resources required to reproduce Figure 5 from the manuscript.

- **Supplementary-Info/**  
  Scripts and resources for generating supplementary figures.
